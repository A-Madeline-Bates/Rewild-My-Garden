If you are interested in anything you have seen on the site and would like to learn more, please have a look at the suggested links below:

If you would like to find out more information about any of the plants recommended on the site:
RHS Find a Plant - Search by plant name, attribute or both to find plant details and a list of suppliers.
Plants for a Future - A chairt which provides information on ecologically sustainable horticulture.

If you are interested in creating a more detailed pollinator calendar for your land:
Bee Kind - Select the flowers you have in your garden to find out your bee kind score.

If you would like to find out more about wildlife projects near you:
Find a Wildlife Trust - The Wildlife Trusts is a movement made up of 46 Wildlife Trusts: independent charities with a shared mission.

If you would like to explore further:
RHS Wildlife Gardening - See all of the RHS's Advice to create a thriving, wildlife-friendly garden.
